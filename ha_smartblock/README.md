# HA SmartBlock Add-on

HA SmartBlock is a Home Assistant add-on that provides a visual Blockly editor for Home Assistant automations.

## Install

Add this repository in the Home Assistant Add-on Store:

```text
https://github.com/seomihyeons/HA_smartblock
```

Then install **HA SmartBlock**, start the add-on, and open the Web UI.

## What It Does

- Loads Home Assistant automations
- Uses entity dropdowns populated from your Home Assistant instance
- Opens automation logic as Blockly blocks
- Exports and copies Home Assistant YAML
- Pushes edited automations back to Home Assistant
- Runs automation analysis
- Supports Task Alt YAML testing

## Image

The add-on uses the pre-built container image published to GHCR:

```text
ghcr.io/seomihyeons/ha_smartblock
```
